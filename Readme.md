# ExpressJS.kr

  Express를 위한 사이트입니다.
  
  http://expressjs.kr

## 빌드

설치:

```
$ npm install -g serve
$ npm install
$ make
$ serve . &
$ open http://localhost:3000
```

다시 빌드하려면 다음 명령어를 입력하십시오.:

```
$ make
```
