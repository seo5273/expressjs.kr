Web Applications

Express는 작고 유연한 node.js 웹 애플리케이션 프레임워크입니다. 하나 혹은 여러 페이지를 제작하거나 하이브리드 웹 애플리케이션을 위한 다양한 기능들을 제공합니다.

APIs

무수한 HTTP 유틸리티 메소드와 Connect 미들웨어를 당신이 원하는 대로 사용 할 수 있습니다. 쉽고 간편하게 탄탄하고 사용하기 편리한 API을 생성할 수 있습니다.

Performance

Express는 node.js의 어려운 기능들을 모두 알 필요가 없도록 웹 애플리케이션을 개발하는데에 있어서 필수적인 기능들을 제공합니다.
